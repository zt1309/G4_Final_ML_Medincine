Lung Cancer Segmentation using Deep Learning 

We presents an automated framework for the detection and segmentation of lung nodules from Computed Tomography (CT) images, specifically designed to support Lung-RADS clinical assessments.

The primary goal is to support radiologists by providing high-precision localization of pulmonary nodules. The system evaluates two distinct deep learning approaches:Mask R-CNN: A two-stage architecture that adds a mask prediction branch to provide precise pixel-wise segmentation.YOLO (v8 & v26): Single-stage detectors optimized for high inference speed and efficient clinical screening.
